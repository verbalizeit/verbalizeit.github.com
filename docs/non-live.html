Tasks

Manage your non-live translation tasks

Get all tasks

Get all task envelopes for all statuses

Resource:

/tasks - GET

Parameters:

api_key (required)Your application API key
date_from (optional)"YYYY-MM-DD"
date_to (optional)"YYYY-MM-DD"

Sample Request (URI Parameters or Request Body OK):

GET
https://api.verbalizeit.com/tasks?api_key=YOUR_API_KEY

Sample Response:

	{
	"id":1,
	"created_at":"2012-09-21T19:10:12-04:00",
	"status":"open"
	}

Get all tasks

Get all task envelopes for a specific status

Resource:

/tasks?status=status - GET

Parameters:

api_key (required)Your application API key
status (required)open, in_progress, in_review, or complete

Sample Request (URI Parameters or Request Body OK):

GET
https://api.verbalizeit.com/tasks?status=open&api_key=YOUR_API_KEY

Sample Response:

	{
		"id":1,
		"created_at":"2012-09-21T19:10:12-04:00",
		"status":"open"
	}

Get task

Get envelope for a specific task

Resource:

/tasks/:task_id - GET

Parameters:

api_key (required)Your application API key
task_id (required)id of the task to retrieve
date_from (optional)"YYYY-MM-DD"
date_to (optional)"YYYY-MM-DD"

Sample Request (URI Parameters or Request Body OK):

GET
https://api.verbalizeit.com/tasks/1234?api_key=YOUR_API_KEY

Sample Response:

	{
	id: 1234,
	status: complete,
	api_key : "YOUR_API_KEY",
	created_date : "YYYY-MM-DD HH:MM:SS"
	task_type: "audio_to_text",
	language : {
			language_from: "Mandarin",
			language_from_dialect : "Dialect",
			language_from_accent  : "Accent",
			language_to: "Mandarin",
			language_to_dialect : "Dialect",
			language_from_accent  : "Accent",
		}
	gender {
		gender_from: "Male"
		gender_to  : "Male"
	}

	asset_from : { 
			file_url: "https://s3.com/file_orig.mp3",
			file_type: mpeg/audio,
			gender: "Male"
			file_size: "10234",
			file_length_in_minutes, "3",
			}
	asset_to : {
			file_url: "https://s3.com/file_dest.mp3",
			file_type: "mpeg/audio",
			gender: "Male"
			file_size: "10234",
			file_length_in_minutes, "3",
			}
	special_instructions : "some instructions",
	}

Create Task

Create a non-live translation task

Resource:

/tasks - POST

Sample Request (Request Body):

POST
https://api.verbalizeit.com/tasks/task_id
	{
		workflow=>"1",
		job_id=>"40",
		asset : {				
				asset_path: "path_to_file_or_url",	
				}
		override_instructions : "some instructions"
		}
	}

Sample Response:

	{
		"id"=>161, 
		"created_at":"2012-09-21T19:10:12-04:00"
	}

Jobs

Manage your non-live translation jobs

Get all jobs

Get all job envelopes

Resource:

/jobs - GET

Parameters:

api_key (required)Your application API key
date_from (optional)"YYYY-MM-DD"
date_to (optional)"YYYY-MM-DD"

Sample Request (URI Parameters or Request Body OK):

GET
https://api.verbalizeit.com/jobs
	{
	api_key : "YOUR_API_KEY",
	date_from : "YYYY-MM-DD",		//optional
	date_to	  : "YYYY-MM-DD",		//optional
	}

Sample Response:

	{
	"id":40,
	"created_at":"2012-09-28T12:39:23-04:00",
	"description":"Job Name #55",
	"summary":{
		"open":0,
		"in_progress":0,
		"in_review":0,
		"complete":0
		}
	}
