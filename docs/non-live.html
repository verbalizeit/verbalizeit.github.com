Create Task

Create a non-live translation task

Resource:

/tasks - POST

Sample Request (URI Parameters or Request Body OK):

POST
https://api.verbalizeit.com/tasks/task_id
	{
	api_key : "MY_API_KEY",
	job_id : "my_job_id",			// optional, response from POST /job/create  
  	task_type: "audio_to_text",
	asset : {
			asset_type : file                    // file or url
			asset_path: "path_to_file_or_url",
			gender : "Male",			// "Male", "Female", "None"
			language : "Mandarin"		// optional			
			}
	language : {
			language_from: "Mandarin",
			language_from_dialect : "Dialect",
			language_from_accent  : "Accent",
			language_to: "Mandarin",
			language_to_dialect : "Dialect",
			language_from_accent  : "Accent",
			}
	gender_to : "Male",
	special_instructions : "some instructions"
	}
	postback : { TBD }					//Oauth token, url
	supporting_resources : { TBD }				// video url, extra docs url(s)
	}

Sample Response:

	{
	api_key : "MY_API_KEY"
	job_id  : ""   		      // null if not set (see jobs documentation)
	task_id : "1234567"
	}
