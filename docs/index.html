The VerbalizeIt API


Getting Started

Integrating VerbalizeIt translations into your application is easy. Quickly access live translators or post translation jobs.

The API is currently in beta. Request API access to get started.


Live Speech-to-Speech Translation

Provide access to live translators directly from your application over either voice or data connections. There are several ways to access a live translator:


SIP - Session Initiation Protocol

SIP can be easily integrated into a browser or mobile app using the Phono library

  1. Post a Live Translation Request with the desired language. The API will respond with a SIP number to dial.
  2. Dial the number with your SIP client.


Telephone - Developer Numbers

Make a phone call to one of our developer numbers. Your application will first need to register the call so we can identify the caller and language.

  1. Post a Live Translation Request with the desired language and the phone number you will be calling from. The API will respond with a telephone number that is expecting the call.
  2. Dial the number with your telephone or phone client.


Telephone - Direct Access

VerbalizeIt provides direct phone numbers to applications that need to dial in to translation services without any HTTP requests. This is especially useful for products used internationally where data connections may be unreliable.

  1. Dial the direct access number with your telephone or phone client.


Batch Translation Tasks

Easily translate your existing content- audio, text, video- into the language of your choice. Upload a variety of standard formats to our system, and receive translated content back.
